<static> find() → {String|null}

Determines the location of the tiapp.xml file. It will search the current directory, and all other directories higher in the view hierarchy, in order. If it does not find a tiapp.xml in any of these directories, null is returned.

Source:
  • tiapp.xml.js, line 23
Returns:

The location of the tiapp.xml file, or null if not found.

Type
String | null
Example
var file = require('tiapp.xml').find();
if (file === null) {
  throw new Error('tiapp.xml not found');
} else {
  // do stuff
}

<static> load(file) → {Tiapp}

Parses the given file as a tiapp.xml file and returns a new Tiapp instance. If a file is not provided, find() will attempt to find one automatically. The file is validated, read, and then parse() is called.

Parameters:
Name Type Argument Description
file String <optional>

Path to the tiapp.xml file
Source:
  • tiapp.xml.js, line 62
Throws:
Must be able to find a tiapp.xml file
Type
TiappError
Returns:

A new instance of {Tiapp} based on the loaded file

Type
Tiapp
Examples
// load with a file path
var tiapp = require('tiapp.xml').load('/path/to/tiapp.xml');
// load without a file path, letting find() try to find it for you
var tiapp = require('tiapp.xml').load();
